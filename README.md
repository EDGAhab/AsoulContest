# 使用方法：
1. 可以按下← →进行移动，也可以使用鼠标拖动
2. 可以按下↑进行跳跃
3. 选取一段文字并且点击右键，可以将其加入到谷歌日历的待办事项中
4. 点击右上角插件的嘉然头像可以更换人物，鼠标风格也会跟着改变
5. 小心！Asoul的成员们非常贪吃，她们会慢慢吃掉你输入到文本框里的一切！（按下↓启用/禁止本功能，当没有东西吃的时候，她会向你眨眨眼）
