# 【Chrome Extention】Asoul桌宠

这么可爱的 向晚/嘉然/贝拉/乃琳/珈乐 不领一只回家吗

![image info](images/Banner.png)

# 使用方法：
1.按下方向键可以移动，也可以用鼠标拖动
2.选取一段文字并且点击右键，可以把他加入到谷歌日历的待办事项中
3.点击右上角插件的嘉然头像可以更换人物，鼠标风格也会跟着改变
